# Strategy — Performance Marketing (Advertising-First)

**Company:** Mirthbound Apps (entity may be Mirthbound Interactive, LLC — confirm on legal/storefront.)

**Objective:** Drive **downloads and intent** (App Store taps, saves, profile visits) for consumer iOS apps via **short-form video**, not thought leadership or ship logs.

**Primary channel:** TikTok (reach + creative testing).  
**Secondary:** Instagram Reels (repurpose winners; slightly different caption/cover norms).

---

## GameTime — advertising-first positioning

**Core promise:** A **personal game OS** on iOS—**what to play**, **what you own**, **what you finished**, and **how you’re doing**—on **real game data (IGDB)**: big-catalog feel with covers, scores, platforms, genres/themes, similar titles, and events **without** a spreadsheet vibe.

**What we are not claiming:** “#1,” “fastest,” “most users,” awards, revenue, fabricated user stats, or review scores unless verified in writing.

**Winning angles (see `docs/GAMETIME.md` for full feature + claim lists):**

- **Relief:** Backlog chaos → seven library states, deep filters, smart “what’s next” rows, randomizer.
- **Discovery:** Rails, unified search, events, recommendations queue (when unlocked)—next obsession **inside** the app.
- **Proof of progress:** Stats, Wrapped-style recap (when gated), badges, journal fields—demo library on camera.
- **Trust:** iCloud + JSON backup narrative where accurate; IGDB attribution in UI when relevant.
- **Speed:** Quick cuts, UI-forward demos, “problem → fix in-app.”

## Hook types (TikTok)

Use **0–2s** pattern interrupts:

- **Contrarian:** “Your backlog shouldn’t live in Notes.”
- **POV:** “POV: you finally organized every iOS game you own.”
- **Before/after:** Messy home screen vs GameTime library (your recordings).
- **Listicle:** “3 signs you need a game library app.”
- **Demo-first:** Start on UI; voiceover catches up by second 3.

---

## CTA rules

- **One primary CTA** per video: App Store search by exact app name **or** “link in bio” if you use a single link hub.
- **Avoid:** “Best,” “guaranteed,” “you will” unless supportable.
- **Prefer:** “Try GameTime on the App Store,” “Search GameTime — Mirthbound.”

---

## Cadence (starter)

- **TikTok:** 3–5 posts/week while learning; batch film 1–2 sessions/week.
- **Reels:** 2–4/week, mostly **repurposes** of TikTok winners (new cover + caption).
- **Review:** 15 min weekly—kill losers, double down on hooks that retain.

---

## TikTok vs Instagram (practical differences)

| Dimension | TikTok | Instagram Reels |
|-----------|--------|-------------------|
| **Discovery** | FYP, sound trends | Explore + followers; hashtags lighter weight |
| **Hook** | Extreme clarity in 1s | Slightly more tolerance for brand on cover frame |
| **Text on screen** | Large, 3–5 words per beat | Same, but test cover title as static hook |
| **Caption** | Short + 0–3 tags | First line = hook repeat; CTA can live line 2 |
| **Safe tone** | Avoid implying platform partnership | Same |

## 4-week starter campaign — GameTime

**Week 1 — Awareness + problem**

- Themes: Notes/screenshot chaos, “where did I save that game?”
- KPI direction: watch time, profile taps (log qualitatively if tools limited).

**Week 2 — Discovery + IGDB**

- Themes: Discover rails, unified search, events carousel, similar games—only labels/flows in shipping build.
- KPI: saves/shares as secondary signals.

**Week 3 — Stats / Wrapped / progress**

- Themes: Stats year filters, highlights, Wrapped when gate allows; badges/journal—**demo library** on screen.
- KPI: comment themes (“sync?” “backup?”) feed FAQ content.

**Week 4 — Habit + CTA test**

- Themes: Now tab + randomizer + smart rows; weekly ritual / backlog triage; test CTAs (search name vs link-in-bio)—no fake conversion claims.
- KPI: same CTA test discipline.
